# WechatHelper
修改自https://github.com/chenliangj2ee/WeChatUtis 感谢此项目作者

支持微信7.0版本 自动查找附近的人，自动回复，自动添加好友，自动拉群,录屏等功能。
主要使用工具 uiautomatorviewer.bat 和android的辅助功能。
